# Agent Learning Project

This repository contains a collection of code examples and tutorials for learning about and building AI Agents. The project is structured into different directories, each focusing on a specific aspect of Agent development.

## Directory Structure

- **`01-openai基础库/`**: This directory contains basic examples for getting started with the OpenAI API using the Python client library. It covers topics like API key setup, making basic API calls, handling streaming responses, and managing conversation history.

- **`P2_提示词优化/`**: This section focuses on prompt engineering techniques. It includes practical examples for tasks such as financial text classification, information extraction into JSON format, and text matching.

- **`P3-langchainRAG开发/`**: This is a comprehensive guide to developing Retrieval-Augmented Generation (RAG) applications using the LangChain framework. It covers:
  - Connecting to different language models.
  - Using various prompt templates.
  - Building chains and using parsers.
  - Managing conversational memory.
  - Loading documents of different formats (CSV, JSON, TXT, PDF).
  - Using in-memory and persistent vector stores (ChromaDB).
  - Constructing retrieval chains.

## How to Use

Browse through the directories to find examples relevant to your interests. Each Python script is a self-contained example that can be run to demonstrate a specific concept. Make sure you have the necessary libraries (like `openai`, `langchain`, etc.) installed.

# Agent学习项目

本仓库包含了一系列用于学习和构建AI Agent的代码示例和教程。项目被划分为不同的目录，每个目录都专注于Agent开发的特定方面。

## 目录结构

- **`01-openai基础库/`**: 此目录包含使用OpenAI API的Python客户端库的入门示例。它涵盖了API密钥设置、基本API调用、处理流式响应和管理对话历史等主题。

- **`P2_提示词优化/`**: 本部分专注于提示词工程技术。它包括金融文本分类、将信息提取为JSON格式以及文本匹配等任务的实际示例。

- **`P3-langchainRAG开发/`**: 这是使用LangChain框架开发检索增强生成（RAG）应用的综合指南。它涵盖了：
  - 连接到不同的语言模型。
  - 使用各种提示词模板。
  - 构建链和使用解析器。
  - 管理对话记忆。
  - 加载不同格式的文档（CSV、JSON、TXT、PDF）。
  - 使用内存和持久化向量存储（ChromaDB）。
  - 构建检索链。

## 如何使用

浏览目录以查找与您兴趣相关的示例。每个Python脚本都是一个独立的示例，可以运行以演示特定的概念。请确保已安装必要的库（如 `openai`、`langchain`等）。
